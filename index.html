Pencil Sketch Program Using OpenCV

This program takes an image and converts it into a pencil sketch using OpenCV and Python. The process involves loading the image, converting it to grayscale, inverting the colors, blurring the image, and finally creating the pencil sketch effect.

Steps Involved:

  1. Loading the image: The image is loaded from a specified path.
  2. Converting to grayscale: The loaded image is converted to a grayscale image.
  3. Inverting the grayscale image: The grayscale image colors are inverted.
  4. Blurring the inverted image: The inverted image is blurred using Gaussian blur.
  5. Creating the pencil sketch: The pencil sketch is created by dividing the grayscale image by the blurred inverted image.
  6. Displaying the images: The original image and the pencil sketch are displayed in separate windows.
  7. Saving the result: The pencil sketch is saved as a PNG file.

Python Code:


import pandas as pd
import cv2
import numpy as np

# Step 1: Loading the image
image_path = 'C:\\Users\\Lakshmi Praneetha\\OneDrive\\Pictures\\Screenshots\\Bangtan\\Wallpapers\\Screenshot (2).png'  # Replace with your image path
image = cv2.imread(image_path)

# Step 2: Converting to grayscale
gray_image = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)

# Step 3: Inverting the grayscale image
inverted_image = cv2.bitwise_not(gray_image)

# Step 4: Blurring the inverted image
blurred_image = cv2.GaussianBlur(inverted_image, (21, 21), sigmaX=0, sigmaY=0)

# Step 5: Creating the pencil sketch
pencil_sketch = cv2.divide(gray_image, 255 - blurred_image, scale=256)

# Step 6: Displaying the images
cv2.imshow('Original Image', image)
cv2.imshow('Pencil Sketch', pencil_sketch)

# Step 7: Saving the result
cv2.imwrite('pencil_sketch.png', pencil_sketch)

# Wait for a key press and close the image windows
cv2.waitKey(0)
cv2.destroyAllWindows()
